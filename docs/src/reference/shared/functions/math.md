- **max**

  - **Type:** `number`

## round(value, precision)

Round a number off to a certain amount of digits after the comma

### Parameters

- **value** - Initial value
  - **Type:** `number`
- **precision** - Maximum amount of digits allowed after the comma
  - **Type:** `number`

### Returns

Rounded value

- **Type:** `number`
